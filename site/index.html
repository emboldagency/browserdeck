1. Backdrop Filter

WebKit needs -webkit-backdrop-filter; older Gecko required a flag. Watch the frosted-glass card.

Frosted Glass

If this looks crisp and blurred behind, the engine supports backdrop-filter natively.

2. Native Date Picker

Every engine renders <input type="date"> differently. Chrome has a calendar dropdown, Firefox has a custom picker, WebKit (non-Safari) often falls back to plain text.

3. Custom Scrollbars

WebKit uses ::-webkit-scrollbar. Firefox uses scrollbar-width/scrollbar-color. Different syntax, different results.

4. Font Rendering & Smoothing

macOS WebKit uses subpixel antialiasing. Windows Chromium uses DirectWrite. Firefox has its own stack. Same font, different weight perception.
